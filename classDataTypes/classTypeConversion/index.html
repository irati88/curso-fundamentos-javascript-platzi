  1. Implícita: El mismo JavaScript lo hace
    • 2+ true
  2. Explícita: Nosotros podemos convertir un tipo de dato en otro
    • String()
    • Number()

Lenguajes de programación

  1. Compilados: El código es traducido antes de la ejecución del programa. Chequea los tipos de datos de manera estática. Ejemplos: C, C++, Rust, Go, Swift
  2. Interpretados: Se va traduciendo a medida que se va ejecutando el código. Chequea los tipos de datos de manera dinámica. Ejemplos: JavaScript, Python, Ruby, PHP
